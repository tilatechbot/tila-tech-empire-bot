# 🚀 INSTALLATION & SETUP GUIDE\n\n## Prerequisites\n\n- **Node.js** >= 18.0.0\n- **npm** >= 9.0.0\n- **Telegram Bot Token** (from @BotFather)\n- **WhatsApp Account** (on your phone)\n\n---\n\n## 📥 Installation Steps\n\n### 1. Clone Repository\n\n```bash\ngit clone https://github.com/tilatechbot/tila-tech-empire-bot.git\ncd tila-tech-empire-bot\n```\n\n### 2. Install Dependencies\n\n```bash\nnpm install\n```\n\n### 3. Setup Environment Variables\n\n```bash\ncp .env.example .env\n```\n\nEdit `.env` with your configuration:\n\n```env\n# WhatsApp Bot Configuration\nOWNER_NUMBER=1234567890@s.whatsapp.net\nOWNER_NAME=TILA TECH\n\n# API Keys\nOPENAI_API_KEY=sk-your-openai-key\nRAPIDAPI_KEY=your-rapidapi-key\nYOUTUBE_API_KEY=your-youtube-key\n\n# Database\nMONGODB_URI=mongodb://localhost:27017/tilatech\n\n# Telegram Integration (IMPORTANT)\nTELEGRAM_BOT_TOKEN=YOUR_TELEGRAM_BOT_TOKEN_HERE\nTELEGRAM_CHAT_ID=YOUR_CHAT_ID_HERE\n\n# Server\nPORT=3000\nNODE_ENV=production\n```\n\n### 4. Get Telegram Bot Token\n\n1. Open Telegram and search for **@BotFather**\n2. Send `/newbot`\n3. Follow the prompts\n4. Copy the **API Token**\n5. Paste in `.env` as `TELEGRAM_BOT_TOKEN`\n\n### 5. Start the Bot\n\n```bash\n# Production\nnpm start\n\n# Development (with auto-reload)\nnpm run dev\n```\n\n---\n\n## 🔗 Telegram Bot Connection\n\nOnce the bot starts, you'll see:\n\n```\n🚀 Server running on port 3000\n📱 Connecting WhatsApp...\n🤖 Initializing Telegram Bridge...\n✅ Telegram Bot Bridge System Ready\n```\n\n### Connect via Telegram:\n\n1. Open Telegram\n2. Search for your bot (the one you created with BotFather)\n3. Send `/start`\n4. Choose connection method:\n   - **🔗 Connect WhatsApp** - QR code method (fastest)\n   - **📲 Pair Phone** - Phone number method\n5. Follow the prompts\n\n---\n\n## ⚙️ Configuration Options\n\n### Bot Settings (config.js)\n\n```javascript\nconfig = {\n  // Bot Settings\n  prefix: '.',              // Command prefix\n  owner: process.env.OWNER_NUMBER,\n  ownerName: 'TILA TECH',\n  botName: 'TILA TECH EMPIRE BOT',\n  botVersion: '2.0.0',\n  \n  // Telegram\n  telegram: {\n    enabled: true,\n    token: process.env.TELEGRAM_BOT_TOKEN,\n    chatId: process.env.TELEGRAM_CHAT_ID,\n  },\n  \n  // Features\n  features: {\n    antilink: true,\n    antibot: true,\n    antispam: true,\n    autoreply: true,\n    economy: true,\n  },\n  \n  // Rate Limiting\n  rateLimit: {\n    enabled: true,\n    commandsPerMinute: 10,\n    messagesPerMinute: 30,\n  },\n};\n```\n\n---\n\n## 📱 WhatsApp Connection Methods\n\n### Method 1: QR Code (Recommended)\n\n```\nUser → /connect\n    ↓\nBot generates QR code\n    ↓\nUser scans with WhatsApp\n    ↓\n✅ Connected!\n```\n\n**Pros:**\n- Fastest method\n- No phone number needed\n- More secure\n\n**Cons:**\n- QR expires in 2 minutes\n- Requires WhatsApp Linked Devices\n\n### Method 2: Phone Number\n\n```\nUser → /pair\n    ↓\nUser enters: +1234567890\n    ↓\nBot sends OTP to WhatsApp\n    ↓\nUser enters 6-digit OTP\n    ↓\n✅ Connected!\n```\n\n**Pros:**\n- No QR code needed\n- More time to verify\n- Works on any device\n\n**Cons:**\n- Slightly slower\n- OTP expires in 5 minutes\n\n---\n\n## 🔍 Verify Installation\n\n### Check Status\n\n```bash\ncurl http://localhost:3000/status\n```\n\n**Response:**\n```json\n{\n  \"status\": \"online\",\n  \"bot\": \"TILA TECH EMPIRE BOT\",\n  \"version\": \"2.0.0\",\n  \"whatsapp\": {\n    \"connected\": \"connected\",\n    \"user\": \"John Doe\"\n  },\n  \"telegram\": {\n    \"enabled\": true,\n    \"connected\": \"connected\"\n  },\n  \"commands\": 50,\n  \"uptime\": \"0h 5m\"\n}\n```\n\n### Health Check\n\n```bash\ncurl http://localhost:3000/health\n```\n\n### List Commands\n\n```bash\ncurl http://localhost:3000/api/commands\n```\n\n---\n\n## 🐛 Troubleshooting\n\n### Telegram Bot Not Responding\n\n**Problem:** Bot doesn't respond to `/start`\n\n**Solution:**\n1. Check `TELEGRAM_BOT_TOKEN` in `.env`\n2. Verify token with @BotFather\n3. Restart bot: `npm start`\n4. Check console for errors\n\n### WhatsApp Not Connecting\n\n**Problem:** WhatsApp QR code not appearing\n\n**Solution:**\n1. Check Baileys is installed: `npm install @whiskeysockets/baileys`\n2. Ensure sessions folder exists\n3. Check internet connection\n4. Try phone number pairing instead\n\n### Port Already in Use\n\n**Problem:** \"Port 3000 already in use\"\n\n**Solution:**\n```bash\n# Use different port\nPORT=3001 npm start\n\n# Or kill process on port 3000\nlsof -ti:3000 | xargs kill -9\n```\n\n### MongoDB Connection Failed\n\n**Problem:** Database won't connect\n\n**Solution:**\n1. Ensure MongoDB is running\n2. Check `MONGODB_URI` in `.env`\n3. Verify connection string format\n4. Check network access\n\n---\n\n## 📊 API Endpoints\n\n| Endpoint | Method | Description |\n|----------|--------|-------------|\n| `/status` | GET | System status |\n| `/health` | GET | Health check |\n| `/api/commands` | GET | List all commands |\n| `/api/whatsapp/info` | GET | WhatsApp info |\n| `/metrics` | GET | System metrics |\n| `/telegram/webhook` | POST | Telegram webhook |\n\n---\n\n## 🚀 Deployment\n\n### Heroku\n\n```bash\n# 1.
